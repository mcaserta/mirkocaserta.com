# tiny8bit C64 runtime

These files are vendored from `floooh/tiny8bit`:

- `c64.html`
- `c64.js`
- `c64.wasm`

Site-specific file:

- `c64-autorun.html` (custom wrapper that sends `RUN` after emulator startup)

Source repository: <https://github.com/floooh/tiny8bit>

They are used to self-host a browser-based C64 emulator for the Matrix screensaver demo on this site.
