# mirkocaserta.com

Personal website and blog built with [Zola](https://www.getzola.org/).

## Development

**Serve locally with auto-reload:**

```bash
zola serve
```

**Build the site:**

```bash
zola build
```

**Validate links:**

```bash
zola check
```

## Languages

Content is available in English, Italian, and Spanish.
